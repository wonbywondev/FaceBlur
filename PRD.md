# 얼굴 블러 서비스 PRD (개인화 얼굴 보호 영상 편집기)

**버전**: 1.0  
**작성일**: 2026.01.16  
**작성자**: Perplexity AI (기획 보조)  
**목적**: 영상에서 본인 얼굴은 그대로 두고 나머지 모든 사람 얼굴을 AI 자동 블러 처리. 
유튜브 숏츠/브이로그 콘텐츠 제작자 대상, 프라이버시 보호+빠른 워크플로 제공.

---

## 1. 비즈니스 목표

- **타겟 유저**: 프리랜서 개발자·콘텐츠 크리에이터 (AI/ML 자동화 선호, MacOS/n8n 사용자)
- **문제 해결**: 기존 툴의 수동 노동(키프레임, 타임라인 스크럽) 제거 → 10분 영상 3~5분 처리
- **수익 모델**: Freemium (기본 무료, Pro: 무제한 처리/배치 $9.99/월)
- **차별화**: "내 사진 업로드 → 자동 세그멘테이션 목록 체크" UI로 완전 자동+사용자 제어

## 3. 기능 요구사항

### 3.1 핵심 기능 (MVP)

1. **영상 업로드 & 자동 분석**
   - 지원: MP4/MOV (최대 10분/500MB)
   - AI: 얼굴 탐지(YOLOv8) + 세그멘테이션(SAM2) + 임베딩(ArcFace)

2. **본인 참조 이미지 업로드**
   - 1~5장 사진 → 자동 얼굴 embedding 추출

3. **세그멘테이션 결과 목록 UI**
타임라인 썸네일 목록
	•	얼굴1 (00:05, 유사도 92%) OFF 기본 □ 블러
	•	얼굴2 (00:12, 유사도 15%) ON 기본 ☑ 블러
	•	얼굴3 (00:20, 유사도 30%) ON 기본 ☑ 블러
일괄: “본인 외 모두 블러” | “미리보기”


4. **블러 처리 & 출력**
- 옵션: Gaussian/Mosaic/Blackout
- 출력: 원본 해상도 다운로드, 임베딩 즉시 삭제 로그

### 3.2 추가 기능 (Post-MVP)
- 배치 처리(n8n 연동)
- MacOS MPS 최적화  
- YouTube Shorts 자동 최적화

## 4. 사용자 여정 (User Flow)
- 1.	업로드: 영상 + 본인 사진 (30초)
- 2.	분석 완료: “5개 얼굴 감지됨” (1~2분)
- 3.	목록 확인/체크 (1분)
- 4.	“처리 시작” → 다운로드 (1분) 총: 3~5분


## 5. UI/UX 가이드라인

- **스타일**: 미니멀 MacOS-like (Notion/VSCode 느낌)
- **키 스크린**: 
  1. 드래그앤드롭 업로드
  2. 프로그레스 바 
  3. 체크박스 그리드 (얼굴 목록)
  4. 미리보기 플레이어
- **접근성**: 유사도 % 표시, 키보드 네비, 다크모드

## 6. 상업화 법적 안전 개발 방식

AI 얼굴 블러 개발 시 GDPR/CCPA/BIPA 준수 필수. 다음 5가지 방식으로 리스크 최소화.

1. **프리트레인드 오픈소스 모델 사용 (Apache/MIT)**
   - YOLOv8-face, MediaPipe Face Detection 등 상용 허용 모델 배포

2. **파인튜닝 시 공개 데이터셋만 사용**  
   - COCO, WIDER FACE 등 공개 데이터로 fine-tune
   - 사용자 데이터 학습 금지

3. **온디바이스 처리 (클라이언트 사이드)**
   - 브라우저/WebAssembly나 앱에서 모델 로드
   - 서버 전송 없음 → GDPR "데이터 최소화" 준수

4. **사용자 동의 기반 + 즉시 삭제 워크플로**
   - 업로드 전 명확한 동의 UI
   - 처리 후 24h 내 영상·임베딩 삭제

5. **하이브리드: 사용자 이미지 세트 학습**
   - 사용자 본인 사진만으로 face embedding 학습 (비저장)
   - 타인 데이터 학습 없음

## 7. 기술 스택 (TRD 참고용)

- **프론트**: React + Canvas (세그멘테이션 오버레이)
- **백엔드**: FastAPI + PyTorch (MPS/GPU 최적화)
- **자동화**: n8n + ffmpeg 
- **호스팅**: Vercel/Render (온디바이스 우선)

## 8. 성공 지표 (KPI)

- 처리 시간: <5분/10분 영상
- 사용자 만족: NPS 80+
- 전환율: 무료→Pro 20%
